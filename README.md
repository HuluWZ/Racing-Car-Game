# Racing-Car-Game

Computer Graphics Course Group Project
Racing Car Game using C++ <GL/glut.h>


# How To Play A Game
  
    SPACE : To Start A Game
    ESC   : To Exit A Game

   UP_ARROW    : Increase Speed
   DOWN_ARROW  : Decrease Speed 
   RIGHT_ARROW : Turn  Right
   LEFT_ARROW  : Turn  Left
